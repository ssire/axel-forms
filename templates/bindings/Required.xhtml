Civility :

The 'required' binding

Goal : makes a group of radio buttons or checkboxes required in a form; compatible with the 'input' plugin of type 'radio' or 'checkbox'

You must follow the validation command convention to allow the validator to extract the field name to print in error message (i.e. the binding host must be a sibling of the field label which must be identified with a class set to label).

Declaration :

  • Parent's host : data-binding="required", data-variable="name"

Usage

In this example there is no value checked by default; the validation will print an error message as long as the user does not select any.
