The "html" plugin

The "html" plugin is a readonly plugin that loads a chunk of HTML.

It can be used in conjunction with other read-only input fields to create editors for viewing documents only. This is useful for instance to generate XTiger XML editors that only differ by the plugins to make one version for statically viewing documents with the read-only input fields and the other version for editing documents with real input fields.

Parameters

None

Basic usage

Use the Input button of the demonstration editor and paste the following XML content to see the 'html' plugin in action :

<Demo>
  <Blob>
    <p xmlns="http://www.w3.org/1999/xhtml">Les <strong>tringles de cistres</strong> tintaient avec un éclat <i>métallique</i></p>
    <p xmlns="http://www.w3.org/1999/xhtml">et sur cette étrange musique les <strong>zingarellas</strong> se levaient</p>
  </Blob>
</Demo>
